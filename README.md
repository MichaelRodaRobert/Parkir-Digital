# 🚗 Aplikasi Pemesanan Slot Parkir - LSP UMDP

Aplikasi berbasis web untuk manajemen dan pemesanan slot parkir secara dinamis dan interaktif. Proyek ini dibangun untuk memenuhi syarat Uji Kompetensi Skema **Pengembang Web (Web Developer)** oleh LSP Universitas Multi Data Palembang dan BNSP.

---

## 🛠️ Spesifikasi Teknologi (Tech Stack)
* **Framework Backend:** Laravel 11 (PHP)
* **Framework Frontend:** Tailwind CSS & Alpine.js (Laravel Breeze)
* **Database:** MySQL
* **Version Control:** Git / GitHub

---

## 👥 Aktor & Fitur Utama

### 1. 👤 Pengguna Parkir
* Registrasi & Login Akun.
* Melihat Status Akun & Edit Profil.
* Melakukan Pemesanan (Booking) Slot Parkir.
* Melihat Status Pemesanan & Konfirmasi Pembayaran.
* Melihat Pengumuman dari Admin.

### 2. ⚡ Admin
* Login ke Aplikasi.
* Verifikasi Pendaftaran Akun (Terima/Tolak).
* Verifikasi Pemesanan Slot Parkir & Pembayaran.
* Manajemen Pengumuman (CRUD: Tambah, Ubah, Hapus).
